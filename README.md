# Letterboxd Follow Checker :)

**English** | [Español](README.es.md)

---
A simple browser-based tool that allows you to see which users you follow on Letterboxd who do not follow you back.
No external APIs, servers or data collection are involved.

## Features
- Displays users you follow who do not follow you back
- Direct links to user profiles
- Sorting options (A-Z, Z-A and the original order)

## How to use
1. Go to your Letterboxd profile
2. Open Dev Tools in your browser  
   - Chrome / Edge: `F12` or `Ctrl + Shift + I` (Windows/Linux)
   - `Cmd + Option + I` (Mac)
3. Go to the **Console** tab
4. Paste the script (follow-checker.js)
5. Press `Enter` and enjoy! :) 

## Notes
- This does **not require login**, but if you are logged in, clicking "View" allows you to unfollow users more easily
- No data is stored, tracked or sent anywhere, everything runs locally in your browser
- Large accounts may take longer to process due to pagination (multiple pages of followers/following)
- This is an unofficial tool and is not affiliated with Letterboxd
- This project is intended for personal and educational use
