# Letterboxd Follow Checker :)  
  
[English](README.md) | **Español**   
  
---  
  
Es una herramienta que funciona con el navegador para ver qué usuarios que sigues en Letterboxd no te siguen de regreso  
No usa APIs, servidores ni recopila tus datos  
  
## Funcionalidades  
- Muestra los usuarios que sigues pero no te siguen  
- Enlaces directo a los perfiles de esos usuarios  
- Opciones para ordenar los usuarios (A-Z, Z-A y por orden original)  
  
## Cómo se usa  
1. Ve a tu perfil de Letterboxd  
2. Abre Herramientas para desarrolladores en tu navegador  
   - Chrome / Edge: `F12` or `Ctrl + Shift + I` (Windows/Linux)  
   - `Cmd + Option + I` (Mac)  
3. Ve a la pestaña **Console**  
4. Pega el script (`follow-checker.js`)  
5. Presiona `Enter` y listo :)   
  
## Notas  
- No es necesario iniciar sesión, pero si estás logueado, al hacer clic en "View" puedes dejar de seguir al usuario más fácilmente si así lo deseas  
- No se almacena nada de tus datos, ni se envía información, todo se lleva a cabo localmente en tu navegador  
- Las cuentas grandes se pueden tardar más por la cantidad de páginas en following/followers  
- Esta herramienta no es oficial y no está asociada a Letterboxd  
- Este proyecto es para uso personal
